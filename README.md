# Beyond the Cubes

Репозиторий инструментов и алгоритмов для построения современных систем финансового планирования и анализа (FP&A). Здесь собраны решения на стыке **Data Engineering**, **Data Science** и **Financial Modeling**.

## Философия

### Основные направления:
- **Алгоритмические аллокации:** Решение проблемы взаимных услуг и циклических ссылок через матричное исчисление и теорию графов.
- **Data-Driven Planning:** Использование статистических методов и ML для прогнозирования и сценарного анализа.
- **Performance Optimization:** Оптимизация расчетов в EPM-системах (Optimacros и др.) через пре-процессинг данных.
